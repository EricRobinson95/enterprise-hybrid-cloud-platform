# Project Charter

# Enterprise Hybrid Cloud Platform

## Project Vision

The Enterprise Hybrid Cloud Platform is a production-style infrastructure designed to host my professional portfolio while demonstrating enterprise networking, cloud engineering, infrastructure automation, and security best practices.

This platform integrates an on-premises home lab with Amazon Web Services (AWS), Microsoft Azure, and Cloudflare to create a secure, scalable, and highly available hybrid cloud environment. The infrastructure is intended to evolve over time as new technologies, projects, and engineering concepts are implemented.

---

# Project Purpose

The purpose of this project is to build and maintain a real-world enterprise infrastructure that showcases practical experience in networking, cloud computing, automation, and modern web application hosting.

Rather than functioning solely as a learning lab, this platform will serve as the production environment for my professional website while demonstrating the planning, deployment, documentation, and operational practices used by enterprise IT organizations.

---

# Project Goals

The project aims to:

- Build a secure enterprise hybrid cloud infrastructure.
- Integrate an on-premises network with AWS and Microsoft Azure.
- Secure public-facing services using Cloudflare.
- Host a modern web application containing my professional portfolio.
- Demonstrate enterprise networking principles and cloud architecture.
- Automate infrastructure deployment using Terraform.
- Implement secure remote administration and hybrid cloud connectivity.
- Monitor infrastructure health, security, and availability.
- Maintain professional documentation throughout the project lifecycle.
- Continuously expand the platform as new technologies are learned.
