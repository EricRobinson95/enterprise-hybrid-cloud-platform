# Enterprise Hybrid Cloud Platform

## Overview

The Enterprise Hybrid Cloud Platform is a production-style hybrid cloud environment designed to host my professional portfolio while demonstrating enterprise networking, cloud engineering, infrastructure automation, and security best practices.

This project integrates an on-premises home lab with Amazon Web Services (AWS), Microsoft Azure, and Cloudflare to create a secure, scalable, and highly available enterprise infrastructure. Every component is designed, deployed, documented, and maintained using the same principles found in modern enterprise environments.

The platform serves two primary purposes:

- Host my professional portfolio, resume, technical documentation, and project showcase.
- Demonstrate real-world enterprise networking and cloud engineering skills through a continuously evolving infrastructure.

---

# Project Goals

- Build a production-style hybrid cloud infrastructure.
- Integrate an on-premises environment with AWS and Microsoft Azure.
- Secure public services using Cloudflare.
- Deploy infrastructure using Infrastructure as Code (Terraform).
- Implement secure connectivity between all environments.
- Apply enterprise security and networking best practices.
- Monitor infrastructure health and performance.
- Document every stage of the design, deployment, and validation process.
